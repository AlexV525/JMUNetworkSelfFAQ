# 集美大学校园网自助 FAQ

这是一个静态网页项目，用于汇总集美大学校园网的常见问题与自助操作指引。

## 内容概览

- 校园网开户
- 客户端使用（Windows/Mac）
- 移动端接入
- 路由器设置
- 宽带绑定
- 常见错误信息
- 联系方式

## 本地查看

1. 克隆仓库
2. 用浏览器直接打开 `index.html`

## 目录结构

- `index.html`：主页面
- `assets/css`：样式文件
- `assets/js`：交互脚本
- `assets/images`：截图资源
- `assets/download`：客户端下载文件
